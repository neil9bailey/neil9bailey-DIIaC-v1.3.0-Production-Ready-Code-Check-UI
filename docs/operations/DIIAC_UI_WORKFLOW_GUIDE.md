# DIIaC UI Workflow Guide (v1.3.0-ui)

This walkthrough describes a standard governance decision flow in the UI.

## 1. Sign In

- Open `https://diiacui.vendorlogic.io`
- Sign in using Entra credentials
- Confirm your role (admin or standard)

## 2. Create Decision Input

- Provide scenario/context
- Add relevant role and evidence details
- Confirm reasoning/policy level as required

## 3. Run Governance Compile

- Submit the workflow
- System runs deterministic governance logic
- LLM synthesis remains constrained by policy and controls

## 4. Review Outputs

Inspect:

- Board report/recommendation
- Ranked vendor options and score gap
- Confidence indicators
- Policy and evidence sections

## 5. Validate Trust Artifacts

Review:

- Execution hash/manifest
- Merkle and ledger references
- Signature presence and key metadata

## 6. Export Decision Pack

- Export for governance review and archive
- Use offline verifier runbook for independent validation when required

## 7. Capture Evidence

Store execution and validation artifacts in release evidence folders for audit continuity.
