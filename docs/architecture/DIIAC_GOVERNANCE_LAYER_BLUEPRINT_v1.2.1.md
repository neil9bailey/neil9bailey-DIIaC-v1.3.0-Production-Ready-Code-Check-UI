# Governance Layer Blueprint (Maintained Reference, v1.3.0-ui)

Note: filename retains `v1.2.1` for historical continuity; this content reflects current runtime behavior.

## Inputs

- User decision context and role evidence
- Business profile constraints
- Policy pack controls
- Optional LLM synthesis (policy-gated)

## Processing Stages

1. Input normalisation and profile checks
2. Policy/control enforcement
3. Deterministic compile and scoring
4. Evidence and trust artifact generation
5. Export/signature packaging

## Outputs

- Board report and recommendation
- Ranked options and confidence signals
- Deterministic manifests and trust data

## Governance Guardrails

- Strict deterministic mode in production
- Required evidence thresholds
- Signature and trust readiness checks
