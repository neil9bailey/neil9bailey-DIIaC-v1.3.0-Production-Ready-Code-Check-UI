# Feature Documentation (Maintained Reference, v1.3.0-ui)

This file keeps its legacy name for continuity, but reflects current feature behavior.

## UI Features

- Entra sign-in and role-aware dashboard
- Governance execution workflow
- Decision pack viewing/export
- Admin and diagnostics surfaces

## Bridge Features

- Entra JWT validation (RS256 production path)
- Group/app-role RBAC mapping
- Governance workflow orchestration
- Copilot-only ingestion mode

## Runtime Features

- Deterministic compile and scoring engine
- Signature, hash, and Merkle outputs
- Trust and verification endpoints
- Audit export support

## Deployment Features

- Dedicated ACA topology for production UI stack
- Shared Key Vault secret references
- Custom domain and TLS for external UI access
